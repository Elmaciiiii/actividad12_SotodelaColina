🎨 Canva con p5.js - Dibujo Interactivo

🎬 Usá el botón para iniciar o detener la animación
🖱 Mové el mouse para dibujar
👆 Hacé clic izquierdo para cambiar de color
🔤 Presioná cualquier tecla para cambiar la forma
🧽 Mantené presionada la tecla B para borrar
🗑️ Presioná la tecla E para borrar todo el dibujo
💾 Botón "Guardar imagen" descarga tu dibujo

⚡ Canva con Processing.js - Animaciones Automáticas

🎨 Hace clic en el canvas para generar formas animadas
🌈 Las formas cambian de color automáticamente
✨ Efectos de rotación y movimiento dinámicos
⚡ Botón "Reiniciar Processing" limpia el canvas
